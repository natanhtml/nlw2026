# devroast

Drop your code, get brutally honest feedback.

## Funcionalidades

- **Cole seu código** - Interface simples para submeter código
- **Roast Mode** - Mode sarcastic analysis
- **Leaderboard** - See the worst code ranked by shame
- **Syntax highlighting** - Code beautifully highlighted

## Como rodar

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
